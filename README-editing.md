# Как править презентацию

Основной текст слайдов лежит здесь:

`src/slides/content.ts`

Один блок в массиве `presentationSlides` — это один слайд. Чтобы добавить новый слайд, скопируйте похожий блок, вставьте его в нужное место и поменяйте текст.

Полезные ссылки при локальном просмотре:

- `http://127.0.0.1:5173/ai-coaching-presentation/` — мини-лендинг
- `http://127.0.0.1:5173/ai-coaching-presentation/#deck` — открыть презентацию
- `http://127.0.0.1:5173/ai-coaching-presentation/#7` — открыть конкретный слайд

Навигация:

- На компьютере: стрелки, пробел, PageUp/PageDown, Home/End.
- На телефоне: свайп влево/вправо и кнопки снизу.
- Полный экран: клавиша `F`.

Картинки для веб-версии лежат в `public/assets`. Оригиналы в корне проекта можно хранить отдельно, но для публикации лучше подключать сжатые версии.
